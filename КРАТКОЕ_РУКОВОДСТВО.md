# 🎯 KILASZLO - Краткое руководство для начинающих

**Язык**: Русский  
**Платформа**: Flutter Web  
**Сложность**: Средняя  
**Время настройки**: 10-15 минут

---

## Что это?

**KILASZLO** - это приложение для общения с искусственным интеллектом (Claude), специально разработанное для людей с ограниченными возможностями.

### Основные возможности:
- 💬 Чат с ИИ на 1000+ тем
- 🔊 Озвучивание ответов ИИ
- 📚 История чатов (сохраняется автоматически)
- 🎨 Большие кнопки для удобного использования
- 📱 Адаптивный дизайн для телефонов и планшетов

---

## 🚀 Быстрый старт (5 шагов)

### Шаг 1: Получить API ключ

1. Откройте https://console.anthropic.com/
2. Создайте аккаунт (если его нет)
3. Нажмите "API keys"
4. Создайте новый ключ
5. **Скопируйте его** (выглядит так: `sk-ant-abc123...`)

### Шаг 2: Добавить ключ в приложение

1. Откройте папку проекта: `C:\dev\projects\kilaszlo`
2. Найдите файл: `lib\services\anthropic_service.dart`
3. Найдите строку:
```dart
static const String _apiKey = 'YOUR_ANTHROPIC_API_KEY';
```
4. Замените на ваш ключ:
```dart
static const String _apiKey = 'sk-ant-ваш_ключ_здесь';
```

### Шаг 3: Установить зависимости

Откройте PowerShell в папке проекта и выполните:

```powershell
cd C:\dev\projects\kilaszlo
flutter pub get
```

### Шаг 4: Запустить приложение

Выполните в PowerShell:

```powershell
flutter run -d web
```

Приложение откроется в браузере 🎉

### Шаг 5: Тестировать

1. Нажмите **"ВЫБРАТЬ ТЕМУ"**
2. Выберите интересующую вас сферу
3. Выберите подсферу
4. Выберите конкретную тему
5. ИИ начнет разговор - нажимайте на предложения или слушайте озвучивание!

## 💡 Примеры использования

### Пример 1: Узнать о Древнем Египте

1. Главный экран → **ВЫБРАТЬ ТЕМУ**
2. Выберите **История**
3. Выберите **Древний мир**
4. Выберите **Древний Египет**
5. ИИ расскажет об Египте
6. Нажмите **ОЗВУЧИТЬ** чтобы услышать
7. Выберите одно из 5 предложений (например "Расскажи про фараонов")
8. ИИ продолжит разговор

### Пример 2: Обсудить квантовую физику

1. Главный экран → **ВЫБРАТЬ ТЕМУ**
2. Выберите **Наука**
3. Выберите **Физика - Современная**
4. Выберите **Квантовая механика**
5. Учитесь новому! 🧠

### Пример 3: Вернуться к старому разговору

1. Главный экран → **ИСТОРИЯ ЧАТОВ**
2. Выберите один из ранее сохраненных чатов
3. Нажмите на чат → продолжить разговор
4. Нажмите на три точки (⋮) → удалить чат

## 🆘 Частые ошибки

### ❌ "API Key not found"
**Решение**: Убедитесь что вы добавили ключ в `anthropic_service.dart`

### ❌ "Connection refused"
**Решение**: Проверьте интернет и что API ключ верный

### ❌ Озвучивание не работает
**Решение**: Проверьте что:
- На компьютере включены колонки
- На мобильнике включен звук
- Браузер позволяет использовать микрофон/звук

### ❌ "Flutter not found"
**Решение**: Установите Flutter с https://flutter.dev/docs/get-started/install

## 📱 На мобильном устройстве

### Android

1. Подключите телефон по USB
2. Включите USB Debug (Настройки → О телефоне → Нажмите версию 7 раз → Режим разработчика → USB Debug)
3. В PowerShell: `flutter run`
4. Приложение установится на телефон

### iPhone

1. Подключите iPhone по USB
2. Выполните: `flutter run`
3. Когда запросит - выберите ваш iPhone
4. Приложение установится

## 💾 Сохранение и история

### Где сохраняется история?

- **На компьютере (веб)**: В Local Storage браузера (автоматически)
- **На Android**: В приватной папке приложения (автоматически)
