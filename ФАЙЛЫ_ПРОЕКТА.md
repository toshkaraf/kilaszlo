## 🎯 КРАТКОЕ ОПИСАНИЕ КАЖДОГО ФАЙЛА

### Основной код

#### 1. **lib/main.dart** (50 строк)
```
Задача: Точка входа приложения
Функции:
- Инициализация Storage сервиса
- Конфигурация Provider
- Запуск приложения
- Установка темы
```

#### 2. **lib/models/theme_data.dart** (1500 строк)
```
Задача: Иерархия всех тем приложения
Содержит:
- 11 категорий (ThemeCategory)
- 40+ подкатегорий (ThemeSubcategory)
- 1000+ отдельных тем (Topic)
- Полное описание каждой темы
```

#### 3. **lib/models/chat.dart** (80 строк)
```
Задача: Модели данных для чатов
Классы:
- ChatMessage - сообщение в чате
- Chat - весь чат целиком
- JSON сериализация для сохранения
```

#### 4. **lib/services/anthropic_service.dart** (150 строк)
```
Задача: Интеграция с Claude AI API
Методы:
- generateAIResponse() - получить ответ ИИ
- generateSuggestedResponses() - 5 предложений
- generateInitialMessage() - первое сообщение
- Обработка ошибок и timeout
```

#### 5. **lib/services/chat_storage_service.dart** (130 строк)
```
Задача: Сохранение и загрузка чатов
Методы:
- getAllChats() - получить все чаты
- saveChat() - сохранить чат
- deleteChat() - удалить чат
- addMessageToChat() - добавить сообщение
- createNewChat() - создать новый чат
```

#### 6. **lib/providers/chat_provider.dart** (200 строк)
```
Задача: Управление состоянием приложения (State Management)
Функции:
- Текущий чат
- История чатов
- Выбранные темы
- Загрузка и ошибки
- Методы для действий пользователя
```

#### 7. **lib/pages/home_page.dart** (150 строк)
```
Задача: Главный экран приложения
Элементы:
- Логотип и заголовок KILASZLO
- Большая кнопка "ВЫБРАТЬ ТЕМУ"
- Большая кнопка "ИСТОРИЯ ЧАТОВ"
- Информационный текст
- Красивый дизайн
```

#### 8. **lib/pages/theme_selector_page.dart** (350 строк)
```
Задача: Трехуровневое меню выбора тем
Уровни:
- 1: Категории (История, Наука, ...)
- 2: Подкатегории (для истории: Древний мир, ...)
- 3: Темы (Древний Египет, Греция, ...)
Особенности:
- Цветовая кодировка
- Плавная навигация
- Описания для каждого уровня
```

#### 9. **lib/pages/chat_page.dart** (400 строк)
```
Задача: Окно чата с ИИ
Функции:
- Отображение сообщений
- Озвучивание текста (Flutter TTS)
- Большие кнопки предложений (5 штук)
- История разговора
- Обработка ввода пользователя
- Индикатор загрузки
```

#### 10. **lib/pages/chat_history_page.dart** (200 строк)
```
Задача: Управление историей чатов
Функции:
- Список всех чатов
- Информация о каждом (дата, количество сообщений)
- Возобновление чата
- Удаление чата с подтверждением
- Форматирование дат
```

